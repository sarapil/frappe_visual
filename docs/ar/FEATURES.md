# المميزات — Frappe Visual

## المميزات المجانية (الطبقة المجتمعية)

### 🗺️ خريطة التطبيق
تصور تفاعلي كامل لهيكل أي تطبيق فريب. يعرض الوحدات كمجموعات، وأنواع المستندات كعقد ملونة، والعلاقات كحواف متحركة.

### 🔗 مستكشف العلاقات
استكشف علاقات DocType بعمق قابل للتعديل. انقر على أي عقدة لعرض جميع الروابط والتبعيات.

### 📊 لوحة المعلومات المرئية
عناصر واجهة قائمة على البطاقات لعرض إحصائيات سريعة عن التطبيق.

### 🧙 معالج القصة المصورة
إنشاء معالجات متعددة الخطوات مع رسوم متحركة وتفرعات منطقية.

### 🌙 الوضع الداكن
دعم كامل للوضع الداكن مع انتقال سلس ومتغيرات CSS مخصصة.

### 📐 دعم RTL
واجهة عربية كاملة من اليمين لليسار مع انعكاس تلقائي للتخطيطات.

### 📐 التخطيطات الأساسية
- القوة الموجهة (fcose)
- التسلسل الهرمي (breadthfirst)

### 🔍 البحث والتصفية
بحث فوري في العقد مع تصفية حسب النوع والوحدة.

### 🗜️ الخريطة المصغرة
نافذة تنقل مصغرة للرسوم البيانية الكبيرة.

---

## المميزات الاحترافية (مدفوعة)

### 📐 التخطيطات المتقدمة
- ELK الطبقي — تخطيط احترافي متعدد الطبقات
- ELK الشجري — عرض شجري منظم
- ELK الضغط — تخطيط مضغوط
- ELK الدائري — تخطيط دائري

### 🪟 النوافذ العائمة
نوافذ قابلة للسحب بنمط سطح المكتب لعرض التفاصيل.

### 🎬 محرك الرسوم المتحركة
- خطوط النمل المتحركة على الحواف
- تأثير النبض المتوهج
- انتقالات متسلسلة عند الدخول والخروج

### 📤 التصدير
- تصدير كصورة SVG متجهية
- تصدير كصورة PNG نقطية

### 🔘 قوائم السياق
قوائم سياق دائرية عند النقر بزر الفأرة الأيمن.

### 🔲 مجموعات مركبة
تجميع العقد في مجموعات قابلة للتوسيع والطي.

### 🏷️ شارات الملخص
عناصر ملخص مدمجة في الرسم البياني.

---

## مميزات المؤسسات

### 🤖 تكامل MCP
خادم Model Context Protocol للتكامل مع مساعدي الذكاء الاصطناعي (مثل Claude).

### 🔌 وصول API كامل
واجهة برمجة تطبيقات REST كاملة للأتمتة والتكامل.

### 🏷️ العلامة البيضاء
إمكانية تخصيص العلامة التجارية (الشعار، الألوان، الاسم).

### ⭐ دعم أولوية
وقت استجابة 24 ساعة مع ساعتين تطوير مخصص شهرياً.
