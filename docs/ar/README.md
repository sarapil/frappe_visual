# Frappe Visual — فريب فيجوال

<div dir="rtl">

## نظرة عامة

**Frappe Visual** هو إطار عمل لتجربة المستخدم مبني على الرسوم البيانية (Graph-based UX) لمنصة فريب. يوفر تصورات تفاعلية لعرض هيكلية التطبيقات والعلاقات بين أنواع المستندات بطريقة بصرية جذابة.

## المميزات الرئيسية

- 🗺️ **خريطة التطبيق** — تصور كامل لهيكل أي تطبيق فريب
- 🔗 **مستكشف العلاقات** — استكشاف تفاعلي لعلاقات DocType
- 🎨 **الوضع الداكن** — دعم كامل مع انتقال سلس
- 📐 **دعم RTL** — واجهة عربية بالكامل من اليمين لليسار
- 📊 **لوحة معلومات مرئية** — عناصر واجهة قائمة على البطاقات
- 🧙 **معالج القصة المصورة** — معالجات متعددة الخطوات مع رسوم متحركة
- 🗜️ **خريطة مصغرة** — تنقل سهل في الرسوم البيانية الكبيرة
- 📤 **تصدير** — SVG و PNG

## التثبيت

```bash
bench get-app --branch main https://github.com/ArkanLab/frappe_visual.git
bench --site mysite install-app frappe_visual
bench build --app frappe_visual
```
