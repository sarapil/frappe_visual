## Application Architecture

### Loading Strategy
1. **fv_bootstrap.js** (~2KB) — loads on every page, provides `frappe.visual` namespace
2. **frappe_visual.bundle.js** (~400KB gzip) — lazy-loaded only when first visual is requested
3. **frappe_visual.bundle.css** — loaded globally for CSS variable support

### Core Architecture
```
frappe.visual.engine() → loads bundle → returns GraphEngine class
frappe.visual.create(opts) → loads + creates GraphEngine instance
frappe.visual.appMap(container, app) → loads + creates AppMap component
```

### Data Flow
```
Component → DataAdapter.fetchXxx() → frappe.xcall('frappe_visual.api.xxx') → Server API → frappe.qb → MariaDB
```
