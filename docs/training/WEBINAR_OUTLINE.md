# Frappe Visual — Webinar Outline

> 45-minute live webinar for developers and consultants

---

## Webinar Title
**"Visual Data Architecture: Transforming Frappe Development with Interactive Graphs"**

---

## Agenda

### 1. The Problem (5 min)
- Data model complexity in large Frappe apps
- Documentation debt and onboarding friction
- Static diagrams become stale instantly
- **Poll**: "How long does it take to onboard a new developer on your ERPNext instance?"

### 2. Introduction to Frappe Visual (5 min)
- What it is: Graph-based UX framework for Frappe
- Technology stack: Cytoscape.js, ELK.js, GSAP
- Open Core model: Free tier + Premium features
- Live on Frappe Cloud Marketplace

### 3. Live Demo (15 min)
- Install on a demo site (pre-recorded backup clip)
- Navigate to Visual Hub
- Explore ERPNext app map
- Deep-dive into Accounts module
- Relationship Explorer for Sales Invoice
- Switch layouts (Force → ELK Layered → Radial)
- Dark mode toggle
- Export as SVG
- **Q&A pause**: "Any questions so far?"

### 4. Developer Integration (10 min)
- Embed AppMap in custom pages
- RelationshipExplorer in dialogs
- Custom node types for domain-specific visualizations
- Server-side API usage
- Storyboard Wizard for onboarding flows

### 5. Premium Features (5 min)
- Advanced ELK layouts
- Floating Windows
- Animation Engine
- MCP Server for AI integration
- White labeling
- Pricing overview

### 6. Q&A (5 min)
- Open floor for questions
- Share links: GitHub, docs, Frappe Cloud listing
