## Title
Frappe Visual — Interactive Graph Visualizations

## Subtitle
See your Frappe app's architecture as interactive node graphs. Explore doctypes, relationships, and modules visually.

## Description

### What is Frappe Visual?
Frappe Visual is a graph-based UX framework that transforms how you understand Frappe applications. Instead of scrolling through doctype lists, explore your entire app architecture as an interactive node graph.

### Key Features
- **App Map** — Visualize any app's complete structure with colored nodes and animated edges
- **Relationship Explorer** — Click any DocType to explore all its links and dependencies
- **9 Layout Algorithms** — Force-directed, hierarchical, ELK enterprise layouts, and more
- **Dark Mode** — Seamless dark theme that follows your system settings
- **RTL Support** — Full Arabic/Hebrew right-to-left interface
- **Export** — Download your graphs as SVG or PNG
- **Embeddable** — Add visualizations to any Frappe page with one line of code

### Who Is It For?
- **Developers** understanding complex codebases
- **Consultants** communicating architecture to clients
- **Admins** auditing system structure
- **Trainers** creating interactive learning materials

### How to Use
1. Install the app
2. Navigate to `/app/visual-hub`
3. Select any app from the dropdown
4. Explore the interactive graph!

### Free vs Premium
- **Free**: App Map, Relationship Explorer, basic layouts, dark mode, RTL, search, minimap
- **Premium**: Advanced ELK layouts, animations, floating windows, export, context menus, API access
