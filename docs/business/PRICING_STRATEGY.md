# Frappe Visual — Pricing Strategy

## Open Core Model

### Free Tier (Community)
**Price**: $0 / forever
**Target**: Individual developers, small teams, evaluation

**Included**:
- AppMap (basic) — visualize any app's structure
- RelationshipExplorer (basic) — explore doctype links
- Visual Dashboard — card-based widgets
- Storyboard Wizard — multi-step wizards
- Dark Mode + RTL support
- Basic layouts (fcose, breadthfirst)
- Search & filter
- Minimap navigation
- Community support (GitHub issues)

### Professional Tier
**Price**: $49/month or $470/year (20% discount)
**Target**: Development teams, consultants, agencies

**Everything in Free, plus**:
- Advanced layouts (ELK.js: layered, tree, stress, radial)
- Floating Windows — desktop-style detail panels
- Animation Engine — ant-lines, pulse glow, stagger effects
- SVG + PNG export
- Context menus (radial right-click)
- ComboGroups + SummaryBadges
- Custom node/edge type API
- Email support (48h response)

### Enterprise Tier
**Price**: $99/month or $950/year (20% discount)
**Target**: Large organizations, ISVs, system integrators

**Everything in Professional, plus**:
- MCP Server integration (AI assistants)
- Full API access for automation
- White labeling (custom branding)
- Priority support (24h response)
- Custom development hours (2h/month)
- Early access to new features

## Frappe Cloud Marketplace

- Listed as subscription-based app
- Free tier auto-enabled (no key needed)
- Premium auto-detected on Frappe Cloud (no key needed)
- Self-hosted users need license key

## License Key Distribution

- Keys generated at: https://arkan.it.com/license
