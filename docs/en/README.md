# Frappe Visual

## Overview

**Frappe Visual** is a graph-based UX framework for Frappe v16. It provides interactive node-graph visualizations that transform how you understand application architecture, doctype relationships, and data models.

## Key Features

- 🗺️ **App Map** — Complete visual representation of any Frappe app structure
- 🔗 **Relationship Explorer** — Interactive DocType relationship graph with depth expansion
- 🎨 **Dark Mode** — Full dark theme support with seamless transitions
- 📐 **RTL Support** — Complete right-to-left interface for Arabic and other RTL languages
- 📊 **Visual Dashboard** — Card-based widgets for quick stats
- 🧙 **Storyboard Wizard** — Multi-step animated wizards with branching logic
- 🗜️ **Minimap** — Navigation overview for large graphs
- 📤 **Export** — SVG and PNG export

## Installation

```bash
bench get-app --branch main https://github.com/ArkanLab/frappe_visual.git
bench --site mysite install-app frappe_visual
bench build --app frappe_visual
```

## Usage

After installation, navigate to:
```
/app/visual-hub
```
