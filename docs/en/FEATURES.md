# Features — Frappe Visual

## Free Features (Community Tier)

### 🗺️ App Map
Interactive full visualization of any Frappe app's structure. Shows modules as groups, doctypes as colored nodes, and relationships as animated edges.

### 🔗 Relationship Explorer
Explore DocType relationships with adjustable depth. Click any node to view all links and dependencies.

### 📊 Visual Dashboard
Card-based widgets displaying quick statistics about the application.

### 🧙 Storyboard Wizard
Create multi-step wizards with animations and branching logic.

### 🌙 Dark Mode
Full dark mode support with seamless CSS variable transitions.

### 📐 RTL Support
Complete Arabic right-to-left interface with automatic layout flipping.

### 📐 Basic Layouts
- Force-directed (fcose)
- Hierarchical (breadthfirst)

### 🔍 Search & Filter
Instant node search with type and module filtering.

### 🗜️ Minimap
Navigation overview window for large graphs.
