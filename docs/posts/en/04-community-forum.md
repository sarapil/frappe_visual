# [Announcement] Frappe Visual — Interactive UI Component Library for Frappe | Open Source

Hi Frappe Community! 👋

We're excited to share **Frappe Visual**, a new open-source developer tools app for Frappe/ERPNext.

## What it does

✅ 30+ Node Types Graph Engine (Cytoscape.js)
✅ 9 Layout Algorithms (ELK.js)
✅ Interactive Kanban Boards
✅ Visual Calendar & Gantt Charts
✅ Relationship Explorer
✅ App Map Visualization
✅ Storyboard Onboarding System
✅ Dark/Light Theme with RTL
✅ Floating Window Components
✅ 3000+ Tabler Icons

## Why we built it

- Frappe forms look plain and basic
- No graph/visualization library for Frappe
- Building dashboards requires custom HTML/JS
- No Kanban/Calendar/Gantt out of the box
- RTL support lacking in UI libraries

We couldn't find a good developer tools solution that integrates natively with ERPNext, so we built one.

## Tech Stack

- **Backend:** Python, Frappe Framework v16
- **Frontend:** JavaScript, Frappe UI, frappe_visual components
- **Database:** MariaDB (standard Frappe)
- **License:** GPL-3.0
- **Dependencies:** frappe_visual, caps, arkan_help

## Installation

```bash
bench get-app https://github.com/sarapil/frappe_visual
bench --site your-site install-app frappe_visual
bench --site your-site migrate
```
