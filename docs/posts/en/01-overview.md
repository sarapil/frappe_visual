# 🎨 Frappe Visual — Interactive UI Component Library for Frappe

> **Interactive UI Component Library for Frappe** — Open source, self-hosted, integrated with ERPNext.

---

## Why Frappe Visual?

Businesses in the Developer Tools space face real challenges:
- ❌ Frappe forms look plain and basic
- ❌ No graph/visualization library for Frappe
- ❌ Building dashboards requires custom HTML/JS
- ❌ No Kanban/Calendar/Gantt out of the box
- ❌ RTL support lacking in UI libraries

**Frappe Visual solves all of this** — natively inside Frappe/ERPNext.

## ✨ Key Features

- ✅ 30+ Node Types Graph Engine (Cytoscape.js)
- ✅ 9 Layout Algorithms (ELK.js)
- ✅ Interactive Kanban Boards
- ✅ Visual Calendar & Gantt Charts
- ✅ Relationship Explorer
- ✅ App Map Visualization
- ✅ Storyboard Onboarding System
- ✅ Dark/Light Theme with RTL
- ✅ Floating Window Components
- ✅ 3000+ Tabler Icons
