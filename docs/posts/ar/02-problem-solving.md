## يعمل مع ERPNext الحالي لديك

Frappe Visual لا يستبدل ERPNext — بل **يُوسّعه**. ثبّته بجانب إعدادك الحالي:

```bash
bench get-app https://github.com/sarapil/frappe_visual
bench --site your-site install-app frappe_visual
```

كل بياناتك وسير عملك وتخصيصاتك الحالية تبقى كما هي.
