# Frappe Visual — User Personas

## 👤 System Administrator
- **Role**: Full system configuration and maintenance
- **Goals**: Configure settings, manage users, monitor system health
- **Pain Points**: Complex configuration, troubleshooting errors
- **Key Screens**: Settings, User Management, System Console

## 👤 Power User
- **Role**: Daily operations and advanced features
- **Goals**: Efficient workflow completion, report generation
- **Pain Points**: Repetitive tasks, data entry bottlenecks
- **Key Screens**: Dashboard, Lists, Reports

## 👤 Regular User
- **Role**: Basic data entry and task completion
- **Goals**: Complete assigned tasks, view relevant information
- **Pain Points**: Navigation complexity, permission restrictions
- **Key Screens**: Assigned tasks, data entry forms

## 👤 Manager / Supervisor
- **Role**: Oversight, approvals, and decision-making
- **Goals**: Team performance tracking, workflow approvals
- **Pain Points**: Lack of visibility, delayed approvals
- **Key Screens**: Dashboard, Approval queue, Reports

## 👤 Guest / External User
- **Role**: Limited access to specific information
- **Goals**: View shared data, submit external requests
- **Pain Points**: Limited access, unclear navigation
- **Key Screens**: Portal, shared documents

## الشخصيات (عربي)

### مدير النظام
- الدور: إعداد وصيانة النظام بالكامل
- الأهداف: تهيئة الإعدادات وإدارة المستخدمين

### المستخدم المتقدم
- الدور: العمليات اليومية والميزات المتقدمة
- الأهداف: إنجاز سير العمل بكفاءة

### المستخدم العادي
- الدور: إدخال البيانات الأساسية وإنجاز المهام
- الأهداف: إكمال المهام المسندة
