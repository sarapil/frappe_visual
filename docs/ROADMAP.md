# Frappe Visual — Roadmap

## Current Version: 0.1.0

### ✅ Delivered
- AppMap with 30+ node types
- Relationship Explorer with depth expansion
- 9 layout algorithms (fcose, breadthfirst, ELK variants, circle, grid, concentric)
- Dark mode with auto-detection
- RTL support with auto-detection
- Minimap navigation
- Visual Dashboard
- Storyboard Wizard
- Floating Windows (GSAP Draggable)
- Animation Engine (ant-lines, pulse, transitions)
- Color System (30+ node types, 12 edge types)
- SVG & PNG export
- License/feature gating (Open Core)
- Frappe Visual Settings DocType
