# Layout Showcase

Demonstrates all **9 layout container components**:

| Component          | Description                         |
| ------------------ | ----------------------------------- |
| **Masonry**        | Pinterest-style auto-packing grid   |
| **Dock**           | macOS-style magnifying dock bar     |
| **GridStack**      | Draggable/resizable dashboard tiles |
| **Bento**          | Apple Bento-style asymmetric grid   |
| **InfiniteScroll** | Lazy-loading endless content        |
| **Sortable**       | Drag-and-drop reorderable lists     |
| **VirtualList**    | Virtualized list for 100K+ items    |
| **StackedLayout**  | Layered card stack with peek        |
| **Resizable**      | Split-pane resizable panels         |

Each layout renders with sample content and interactive resize/drag handles.
