# Navigation Demo

Demonstrates all **9 navigation/wayfinding components**:

| Component          | Description                             |
| ------------------ | --------------------------------------- |
| **CommandBar**     | ⌘K command palette with fuzzy search    |
| **FloatingNav**    | Collapsible floating action menu        |
| **PageTransition** | Animated page-to-page transitions       |
| **BackToTop**      | Scroll-to-top floating button           |
| **NavRail**        | Vertical navigation rail (Material 3)   |
| **AnchorNav**      | In-page section anchors with scroll spy |
| **TabBar**         | Horizontal tab navigation               |
| **BottomNav**      | Mobile-style bottom navigation bar      |
| **SpeedDial**      | Expandable floating action button       |
