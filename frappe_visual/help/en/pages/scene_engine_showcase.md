# Scene Engine Showcase

Demonstrates the **18-component Immersive SVG Scene Engine** — animated workspace dashboards where KPIs appear as picture frames, documents scatter on a desk, and books line a shelf.

## Presets Demonstrated

| Preset                  | Best For                   |
| ----------------------- | -------------------------- |
| **scenePresetOffice**   | Business/CRM dashboards    |
| **scenePresetLibrary**  | Knowledge/documentation    |
| **scenePresetClinic**   | Healthcare workspaces      |
| **scenePresetWorkshop** | Construction/manufacturing |
| **scenePresetCafe**     | Hospitality/restaurant     |

## Core Components

SceneEngine, SceneRoom, SceneFrame, SceneDesk, SceneDocument, SceneShelf, SceneBoard, SceneWidget, SceneLighting, SceneDataBinder, SceneRefresher, SceneNavigator, SceneExporter, and 5 preset generators.
