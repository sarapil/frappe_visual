# Visual Playground

Interactive **component sandbox** for testing and experimenting with Frappe Visual components in real-time.

## Features

- Live code editor with syntax highlighting
- Instant component preview pane
- Preset examples for all 307+ components
- Configuration panel to adjust props and options
- Export component code snippets
- Performance profiling for custom configurations
