# Data Viz Gallery

Interactive showcase of all **9 data visualization components** powered by ECharts:

| Component     | Description                               |
| ------------- | ----------------------------------------- |
| **Heatmap**   | Activity/intensity heatmap (GitHub-style) |
| **Sparkline** | Inline mini-charts for cards and tables   |
| **Radar**     | Multi-axis comparison charts              |
| **Funnel**    | Sales/conversion funnel visualization     |
| **Treemap**   | Hierarchical area proportions             |
| **Donut**     | Ring/pie charts with center labels        |
| **DataCard**  | KPI cards with trend indicators           |
| **Area**      | Stacked area time-series charts           |
| **Sankey**    | Flow/distribution diagrams                |

Each component displays with live data and interactive controls to adjust parameters.
