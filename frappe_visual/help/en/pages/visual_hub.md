# Visual Hub

The **Visual Hub** is the main entry point for Frappe Visual. It displays an interactive App Map showing all installed Frappe applications, their modules, DocTypes, and relationships as a navigable graph.

## Features

- **App Map Generator**: Select any installed app to generate a full module→DocType→relationship graph
- **ERD Explorer**: View Entity Relationship Diagrams for any DocType cluster
- **Component Gallery**: Quick-access cards for all 307+ visual components
- **Quick Stats**: Live DocType counts and workspace sparklines
- **Layout Switching**: Toggle between Force, Hierarchical, Circular, Grid, and 5 ELK algorithms

## Usage

1. Navigate to **Desk → Frappe Visual → Visual Hub** or use the /desk URL
2. Select an app from the dropdown to generate its map
3. Click any node to explore its relationships
4. Use the layout selector to change graph algorithms
5. Export as SVG/PNG using the toolbar buttons
