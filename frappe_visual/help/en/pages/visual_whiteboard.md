# Visual Whiteboard

A **freeform drawing and diagramming canvas** for creating flowcharts, architecture diagrams, mind maps, and annotations. Saved as FV Visual Asset records.

## Features

- Freeform drawing with pen, shapes, text, and connectors
- Snap-to-grid and alignment guides
- Multiple layer support
- Collaborative editing (when real-time is enabled)
- Save/load from FV Visual Asset
- Export as SVG, PNG, or PDF
- Embed Frappe data nodes (DocType cards) in diagrams
