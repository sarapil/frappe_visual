# Visual Templates Demo

Showcases all **8 page templates** available in Frappe Visual:

1. **Dashboard Template** — KPI cards + charts grid
2. **ERD Template** — Entity Relationship Diagram page
3. **Workflow Template** — Business process visualization
4. **Tree Template** — Hierarchical data display
5. **Wizard Template** — Multi-step form/onboarding
6. **Kanban Workspace** — Drag-and-drop board
7. **Timeline Template** — Activity/event timeline
8. **App Overview** — Full application showcase

Each template can be generated with a single function call and customized via configuration objects.
