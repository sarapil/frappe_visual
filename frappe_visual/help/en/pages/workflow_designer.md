# Workflow Designer

A visual **drag-and-drop workflow builder** for designing Frappe Workflow states and transitions.

## Features

- Visual node editor for workflow states (with colors and actions)
- Drag connections between states to create transitions
- Auto-generates Frappe Workflow JSON from the visual design
- Save/load workflow designs as FV Visual Assets
- Export workflow diagrams as SVG/PNG for documentation
- Supports all Frappe workflow actions: Approve, Reject, Review, Cancel
