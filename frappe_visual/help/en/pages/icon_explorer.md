# Icon Explorer

Browse, search, and preview all **5,000+ Tabler Icons** available in Frappe Visual.

## Features

- Full-text search with instant filtering
- Category-based browsing (Design, Development, Communication, etc.)
- Size preview (sm, md, lg, xl)
- Color customization preview
- One-click copy: icon name, CSS class, or `frappe.visual.icons.render()` code
- Recently used icons tracking
