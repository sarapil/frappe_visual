# المستخدم المتقدم FV — دليل الدور

يمنح دور **المستخدم المتقدم FV** صلاحية كاملة لجميع المكونات المرئية بالإضافة إلى إمكانيات التصدير وتبديل التخطيط. مثالي للمحللين والمطورين وقادة الفرق.

## الصلاحيات (14)

| الصلاحية             | الوصف                         |
| -------------------- | ----------------------------- |
| `FV_view_visual_hub` | الوصول إلى لوحة المركز المرئي |
| `FV_use_app_map`     | تصفح خرائط التطبيق التفاعلية  |
| `FV_use_erd`         | عرض مخططات العلاقات           |
| `FV_use_storyboard`  | التنقل عبر العروض المصورة     |
| `FV_use_kanban`      | التفاعل مع لوحات كانبان       |
| `FV_use_calendar`    | عرض التقويم المرئي            |
| `FV_use_gantt`       | عرض الجداول الزمنية           |
| `FV_use_map`         | استكشاف الخرائط الجغرافية     |
| `FV_use_gallery`     | تصفح معارض الصور              |
| `FV_use_tree`        | التنقل في العرض الهرمي        |
| `FV_export_svg`      | تصدير الرسوم كملفات SVG       |
| `FV_export_png`      | تصدير الرسوم كصور PNG         |
| `FV_change_layout`   | التبديل بين محركات التخطيط    |
| `FV_view_statistics` | عرض إحصائيات التطبيق          |

## ما يمكنك فعله (بالإضافة إلى العارض)

- **تصدير** أي مكون مرئي كملف SVG أو PNG
- **تبديل التخطيطات** — القوة الموجهة، الهرمي، ELK، Dagre، الدائري
- **النسخ إلى الحافظة** — نسخ الرسم البياني بنقرة واحدة للعروض التقديمية
- **إنشاء ملفات PDF** — تصدير لوحات المعلومات كمستندات PDF
- **حفظ العروض** — إنشاء فلاتر محفوظة للوصول السريع
- **استخدام منشئ السمات** — معاينة سمات مرئية مختلفة

## ما لا يمكنك فعله

- ❌ تعديل إعدادات Frappe Visual
- ❌ إدارة الإعدادات على مستوى التطبيق

## اختصارات التصدير

| الإجراء         | الاختصار                                             |
| --------------- | ---------------------------------------------------- |
| تصدير كـ PNG    | `frappe.visual.exportPNG(element)`                   |
| تصدير كـ SVG    | `frappe.visual.exportSVG(element)`                   |
| تصدير كـ PDF    | `frappe.visual.exportPDF(element, { format: "a4" })` |
| نسخ إلى الحافظة | `frappe.visual.exporter.clipboard(element)`          |
