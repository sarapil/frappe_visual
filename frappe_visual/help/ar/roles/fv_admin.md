# مدير FV — دليل الدور

يوفر دور **مدير FV** تحكماً كاملاً في Frappe Visual بما في ذلك جميع المكونات المرئية والتصدير وتبديل التخطيط وإدارة إعدادات النظام.

| الصلاحية             | الفئة | الوصف                    |
| -------------------- | ----- | ------------------------ |
| `FV_view_visual_hub` | وحدة  | الوصول إلى المركز المرئي |
| `FV_use_app_map`     | وحدة  | تصفح خرائط التطبيق       |
| `FV_use_erd`         | وحدة  | عرض مخططات العلاقات      |

## ميزات المدير الحصرية

### إدارة الإعدادات

انتقل إلى **إعدادات Frappe Visual** لتكوين:

- **المحسنات التلقائية** — تفعيل/تعطيل محسنات النماذج والقوائم ومساحات العمل
- **السمة الافتراضية** — تعيين السمة المرئية الافتراضية لجميع المستخدمين
- **لوحة الألوان** — اختيار نظام الألوان الافتراضي للمخططات
- **تلميحات ثنائية اللغة** — تفعيل/تعطيل المساعدة عند التمرير عربي↔إنجليزي
- **الأداء** — تكوين مدة الذاكرة المؤقتة وحدود التحميل الكسول

### إدارة الإضافات

```javascript
// تسجيل إضافة مخصصة
frappe.visual.registerPlugin({
  name: "my-custom-plugin",
  app: "my_app",
  version: "1.0.0",
  components: { MyWidget: MyWidgetClass },
});
```

### مراقبة الأداء

```javascript
// تفعيل شاشة الأداء للتصحيح
frappe.visual.perf.enable();
frappe.visual.perf.showHUD(); // يعرض FPS, CLS, LCP, الذاكرة
frappe.visual.perf.report(); // الحصول على لقطة كاملة للمقاييس
```

## أفضل الممارسات

1. **اختبر السمات** على التخطيطات LTR و RTL قبل التوزيع
2. **راقب الأداء** بعد تفعيل إضافات أو محسنات جديدة
3. **راجع صلاحيات CAPS** عند إضافة أعضاء جدد للفريق
4. **صدّر الإعدادات** قبل الترقيات الكبيرة كنسخة احتياطية
5. **تحقق من ساحة المكونات** (`/app/visual-playground`) للتأكد من عرض جميع المكونات بشكل صحيح
