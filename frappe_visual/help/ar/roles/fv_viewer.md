# عارض FV — دليل الدور

يوفر دور **عارض FV** صلاحية قراءة فقط لجميع المكونات المرئية في Frappe Visual. هذا هو الدور الافتراضي لأعضاء الفريق الذين يحتاجون لاستكشاف البيانات بصرياً دون تعديل الإعدادات.

## الصلاحيات (11)

| الصلاحية             | الوصف                               |
| -------------------- | ----------------------------------- |
| `FV_view_visual_hub` | الوصول إلى لوحة المركز المرئي       |
| `FV_use_app_map`     | تصفح خرائط التطبيق التفاعلية        |
| `FV_use_erd`         | عرض مخططات العلاقات بين الكيانات    |
| `FV_use_storyboard`  | التنقل عبر العروض التقديمية المصورة |
| `FV_use_kanban`      | التفاعل مع لوحات كانبان             |
| `FV_use_calendar`    | عرض التقويم المرئي                  |
| `FV_use_gantt`       | عرض الجداول الزمنية بأسلوب جانت     |
| `FV_use_map`         | استكشاف الخرائط الجغرافية           |
| `FV_use_gallery`     | تصفح معارض الصور                    |
| `FV_use_tree`        | التنقل في العرض الهرمي              |
| `FV_view_statistics` | عرض إحصائيات التطبيق                |

## ما يمكنك فعله

- **عرض** جميع المكونات المرئية — الرسوم البيانية، مخططات ERD، الخرائط، الجداول الزمنية
- **التنقل** في المركز المرئي وجميع مساحات العمل
- **التفاعل** مع لوحات كانبان والتقويمات والأشجار
- **عرض** لوحات الإحصائيات والتحليلات
- **تصفح** العروض التقديمية ومحتوى الإعداد الأولي

## ما لا يمكنك فعله

- ❌ تصدير الرسوم كملفات SVG أو PNG
- ❌ تغيير محركات التخطيط
- ❌ تعديل إعدادات Frappe Visual
- ❌ إدارة السمات أو التكاملات

## البدء

1. انتقل إلى **المكتب → Frappe Visual** من شبكة أيقونات التطبيقات
2. انقر على **المركز المرئي** لعرض اللوحة الرئيسية
3. استخدم الشريط الجانبي لاستكشاف **خريطة التطبيق**، **ERD**، **كانبان** وغيرها
4. انقر على أيقونة **❓** في أي صفحة للحصول على مساعدة سياقية
