# التقارير والتحليلات — Frappe Visual

يعزز Frappe Visual قدرات التقارير في Frappe بتمثيلات بصرية للبيانات.

## مكونات التقارير المتاحة

| المكون           | الاستخدام                   | الوصف                          |
| ---------------- | --------------------------- | ------------------------------ |
| **خريطة حرارية** | `frappe.visual.heatmap()`   | خريطة نشاط/كثافة (بنمط GitHub) |
| **مخطط شرارة**   | `frappe.visual.sparkline()` | مخططات مصغرة للوحات المعلومات  |
| **دائري**        | `frappe.visual.donut()`     | تصور البيانات النسبية          |
| **مساحي**        | `frappe.visual.area()`      | تحليل الاتجاهات الزمنية        |
| **رادار**        | `frappe.visual.radar()`     | مقارنة متعددة الأبعاد          |
| **قُمعي**        | `frappe.visual.funnel()`    | تحليل المسار/التحويل           |
| **خريطة شجرية**  | `frappe.visual.treemap()`   | نسب البيانات الهرمية           |
| **سانكي**        | `frappe.visual.sankey()`    | تصور التدفق/الاتصالات          |
| **بطاقة بيانات** | `frappe.visual.dataCard()`  | بطاقات مؤشرات الأداء           |

## متطلبات الصلاحيات

يتطلب عرض الإحصائيات صلاحية **FV_view_statistics**.
