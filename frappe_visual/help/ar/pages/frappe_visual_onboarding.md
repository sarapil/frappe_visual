# معالج إعداد Frappe Visual

جولة إرشادية للمستخدمين الجدد تغطي جميع إمكانيات Frappe Visual في 16 خطوة: نظرة عامة، المحسنات التلقائية، المركز البصري، محرك الرسوم، تصور البيانات، محرك المشاهد، التنقل، التخطيطات، الرسوم المتحركة، السبورة، السرد البياني، مُنشئ السمات، الأيقونات، التصدير، الإعدادات، والأدوار.
