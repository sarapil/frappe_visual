# الإعدادات والتكوين — Frappe Visual

تتم إدارة جميع إعدادات Frappe Visual من خلال نوع المستند **Frappe Visual Settings**.

## الوصول إلى الإعدادات

1. انتقل إلى **Frappe Visual Settings** من شريط البحث، أو
2. اذهب إلى `/desk#Form/Frappe Visual Settings`

## خيارات التكوين

### الترخيص

- **مفتاح الترخيص**: مفتاح ترخيص Frappe Visual (للميزات المتقدمة)
- **حالة الترخيص**: يتم التحقق تلقائياً (نشط / منتهي / تجريبي)

### المظهر الافتراضي

- **المظهر الافتراضي**: اختر من سجلات FV Theme المتاحة
- **الوضع الداكن**: تفعيل/تعطيل الوضع الداكن عالمياً

### الأداء

- **تحميل كسول للمكونات**: تحميل المكونات عند الطلب (مُوصى: مُفعّل)
- **تفعيل المحسّنات التلقائية**: تبديل التحسين التلقائي للنماذج/القوائم/مساحات العمل
- **مدة التخزين المؤقت**: مدة تخزين البيانات المرئية مؤقتاً (افتراضي: 300 ثانية)

## متطلبات الصلاحيات

تتطلب إدارة الإعدادات صلاحية **FV_manage_settings**.
