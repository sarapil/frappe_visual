# أداة لوحة المعلومات FV

**أداة لوحة المعلومات FV** هي جدول فرعي من **لوحة المعلومات FV**. يمثل كل صف أداة مرئية واحدة يتم عرضها داخل شبكة لوحة المعلومات.

## # widget_type

نوع العرض المرئي لهذه الأداة. الأنواع المدعومة:

| النوع          | الوصف                                      |
| -------------- | ------------------------------------------ |
| `number_card`  | قيمة مؤشر أداء رئيسي مع مؤشر الاتجاه     |
| `bar`          | رسم بياني شريطي عمودي أو أفقي             |
| `line`         | رسم بياني خطي مع سلسلة زمنية              |
| `donut`        | رسم بياني دائري للنسب                      |
| `heatmap`      | مصفوفة ملونة لبيانات الكثافة               |
| `sparkline`    | رسم بياني مصغر للاتجاه                     |
| `kanban`       | عرض لوحة كانبان                            |

## # widget_config

إعدادات JSON للأداة. لكل نوع أداة مخطط خاص:

```json
{
  "label": "الإيرادات الشهرية",
  "doctype": "Sales Invoice",
  "aggregate": "sum",
  "field": "grand_total",
  "filters": { "status": "Paid" }
}
```

## # width

امتداد الأعمدة في شبكة لوحة المعلومات (١–١٢). يتحكم في الحجم الأفقي.

## # height

امتداد الصفوف في شبكة لوحة المعلومات. يتحكم في الحجم العمودي.

## # sort_order

ترتيب العرض داخل لوحة المعلومات. الأرقام الأصغر تظهر أولاً.
