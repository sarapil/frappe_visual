# إعدادات Frappe Visual

لوحة التحكم المركزية لتطبيق Frappe Visual. يمكن فقط للمستخدمين الذين لديهم صلاحية `FV_manage_settings` تعديل هذه الإعدادات.

## # enable_auto_enhancers

مفتاح رئيسي لجميع المحسّنات التلقائية (النماذج، القوائم، مساحات العمل). عند التعطيل، تتوفر مكونات Frappe Visual فقط عبر الاستدعاء المباشر.

## # enable_form_enhancer

يضيف تلقائياً شريط إحصائيات ورسوم بيانية للعلاقات وأزرار وصول سريع لكل نموذج.

## # enable_list_enhancer

يضيف أوضاع عرض البطاقات/كانبان/الجدول الزمني لكل قائمة.

## # enable_workspace_enhancer

يثري مساحات العمل بعدادات حية ورسوم مصغرة وتأثيرات بصرية.

## # enable_bilingual_tooltip

عرض تلميحات ثنائية اللغة (عربي↔إنجليزي) على جميع العناصر.

## # default_theme

اختيار السمة البصرية النشطة من سجلات FV Theme المتاحة.

## # default_chart_palette

لوحة الألوان الافتراضية لجميع مكونات الرسوم البيانية.

## # license_key

مفتاح الترخيص المميز للميزات المتقدمة (محرك المشاهد، السرد البياني، السبورة التعاونية).

## # default_layout_engine

محرك التخطيط الافتراضي: Force, Hierarchical, Circular, Grid, ELK.

## # max_graph_nodes

الحد الأقصى لعدد العقد في التصورات البيانية. الافتراضي: 500.

## # cache_ttl

مدة صلاحية ذاكرة التخزين المؤقت بالثواني. الافتراضي: 300.

## # enable_premium_features

تفعيل الميزات المميزة التي تتطلب مفتاح ترخيص صالح.
