# سمة FV

تُعرّف **FV Theme** سمة بصرية كاملة تتضمن خصائص CSS مخصصة، ولوحات ألوان المخططات، وتجاوزات تنسيق المكونات. يمكن أن تكون سمة واحدة فقط نشطة (افتراضية) في كل مرة.

## # theme_name

معرّف فريد للسمة (مثل: "arkan-default"، "midnight-blue").

## # title

اسم العرض المقروء للسمة.

## # css_variables

كائن JSON يربط أسماء خصائص CSS المخصصة بقيمها.

## # chart_palette

مصفوفة JSON من ألوان hex تُستخدم كلوحة ألوان افتراضية للمخططات.

## # is_default

سمة واحدة فقط يمكن أن تكون الافتراضية. تعيين سمة جديدة كافتراضية يلغي تنشيط السمة السابقة تلقائياً.

## # status

السمات النشطة متاحة للاختيار. السمات غير النشطة مخفية لكنها محفوظة للاستخدام المستقبلي.

## # preview_image

لقطة شاشة أو صورة مصغرة للسمة مطبقة على لوحة معلومات نموذجية.
