# الأصول البصرية FV

يخزن **FV Visual Asset** التكوينات البصرية الدائمة — حالات السبورة المحفوظة، فصول السرد البياني، تخطيطات الرسوم المخصصة، وقوالب لوحات المعلومات القابلة لإعادة الاستخدام.

## أنواع الأصول

| النوع                  | الوصف                                          |
| ---------------------- | ---------------------------------------------- |
| **Story**              | فصول السرد البياني مع كشف المخططات عند التمرير |
| **Whiteboard**         | رسم حر/مخططات محفوظة (SVG + بيانات وصفية)      |
| **Graph Layout**       | تخطيطات محرك الرسوم المخصصة                    |
| **Dashboard Template** | قوالب لوحات معلومات قابلة لإعادة الاستخدام     |

## # title

اسم مقروء للأصل البصري.

## # asset_type

نوع المحتوى البصري: Story, Whiteboard, Graph Layout, أو Dashboard Template.

## # asset_data

بيانات JSON تحتوي على التكوين الكامل للأصل. يتم التحقق منها عند الحفظ.

## # is_public

عند التفعيل، يكون الأصل مرئياً لجميع المستخدمين. عند التعطيل، يراه المالك فقط.

## # reference_doctype

ربط اختياري بنوع مستند محدد.

## # reference_name

ربط اختياري بسجل محدد.
